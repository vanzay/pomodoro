## Usage

```sh
# Help
pomodoro --help

# Example
pomodoro --focus 50 --short-break 10 --long-break 30
```

### Keyboard shortcuts
- Space - Pause/Resume
- Tab - Skip
- Esc, Ctrl+c - Exit
